## Installation

Clone project

```bash
  git clone https://github.com/trkgrn/auction-webapp-with-websocket
```

Go to the project directory

```bash
  cd auction-webapp-with-websocket
```

Install project dependencies

```bash
  docker-compose up
```



  
## Usage

- Backend

  `http://localhost:8080/`

- Frontend

  `http://localhost:4200/`
